# Changelog

All notable changes to this project will be documented in this file.

## 0.7.0 - 2026-06-20

- Added an IndexedDB learning database.
- Added automatic migration from legacy browser storage.
- Added a full-page contextual spelling review interface.
- Added FSRS spaced-repetition scheduling.
- Added dynamic daily new-word recommendations.
- Added hint-aware automatic review ratings.
- Added contextual retry for incorrect answers.
- Added popup dashboard, library management, and settings.

## 0.6.0 - 2026-06-20

- Added the 80,000-entry ECDICT core dictionary.
- Added inflected-form to lemma mappings.
- Fixed word-chip click handling.
- Added Chinese definitions, pronunciation, and contextual card storage.

## 0.5.0 - 2026-06-20

- Focused subtitle support on Bilibili-generated English subtitles.
- Added player request interception and API fallbacks.
- Removed hard-subtitle OCR support.
